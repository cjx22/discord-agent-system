# Discord Agent System

An event-driven Discord bot built in Python with a modular architecture.
Designed to support persistent state, background tasks, and future AI agents.

## Features
- Async, event-driven architecture using discord.py
- Modular command and service layers
- Environment-based configuration
- Designed for future AI integration

## Tech Stack
- Python 3.10+
- discord.py
- SQLite (planned)
- LLM integration (planned)
